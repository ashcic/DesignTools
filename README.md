# Design Tools for Grasshopper

Design Tools is a set of high-performance Grasshopper components created to make the industrial design workflow fast and flexible.

The componenets themselves are only half of the story, the other aspect is WorkFlow.  Getting that speed and agility depends on understanding how to build.  Design Tools makes that process super simple and is a great entry point to exploring all the amzing things that you can do in Grasshopper.

---

### Getting Started

Design Tools works for either Windows or Mac, Rhino 8.  The best place to get them from is Rhino's inbuilt **Package Manager**.  This will ensure everything is located where it should be and can help you always be up-to-date with the latest and greatest version.
