# Centre Point

<img width="200" height="200" alt="Centre Point component icon" src="../../../images/Centre_Point_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
查找输入几何体的中心点，可以单独查找每个几何体的中心，也可以将它们合并为一个整体并查找整体中心点。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**Union**  
将几何体视为一组并返回一个点

**Individual**  
将几何体视为单独对象并返回多个点
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Geometry**  
主要几何体
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Centers**  
几何体的中心点
</div>

</div>
