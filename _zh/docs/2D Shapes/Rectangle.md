# Rectangle

<img width="200" height="200" alt="Rectangle component icon" src="../../images/Rectangle_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
创建具有可自定义尺寸和角样式的矩形。您可以在尖角、倒角或各种圆角（圆弧）和混合角之间选择。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**C2 Corners**  
每个角的平滑混合半径

**Arc Corners**  
每个角的简单圆弧半径

**Chamfered Corners**  
平边代替圆弧

**C2 Arc Corners**  
在每个角产生模仿圆弧的 C2 平滑半径
</div>

<div class="component-box box-in" markdown="1">
## 输入

**X**  
X 尺寸

**Y**  
Y 尺寸

**Radii**  
最多可添加四个不同半径值以实现不同大小的角

**Blends**  
控制角与边的混合程度（通常值在 0.0 - 1.0 之间）。也可为每个角添加最多 4 个输入以实现不同的混合配置。
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curves**  
作为单独曲线的矩形

**Joined**  
作为连接曲线的矩形

**Points**  
角点

**Notes**  
如何使用此工具的说明
</div>

</div>
