# Pill

<img width="200" height="200" alt="Pill component icon" src="../../../images/Pill_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
基于 X 和 Y 尺寸创建胶囊或跑道形状。使用“Orient”调整放置位置，这对于构建与角对齐的细节很有用。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**Smooth**  
在尖锐几何角和光滑连续曲线之间切换

**Pivot Width**  
如果选中，X 为两个枢轴点之间的距离
如果未选中（默认），X 为总宽度
</div>

<div class="component-box box-in" markdown="1">
## 输入

**X**  
X 尺寸

**Y**  
Y 尺寸

**Orient**  
胶囊的方向
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curves**  
单独的曲线

**Joined**  
连接的曲线

**Pivots**  
圆弧中心的枢轴点
</div>

</div>
