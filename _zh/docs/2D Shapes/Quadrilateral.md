# Quadrilateral

<img width="200" height="200" alt="Quadrilateral component icon" src="/DesignTools/images/Quadrilateral_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
创建平行四边形、梯形、菱形或风筝形。边的角度可以通过角度输入或内缩输入来设置。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**Parallelogram**  
创建平行四边形

**Trapezium**  
创建梯形

**Rhombus**  
创建菱形

**Kite**  
创建风筝形或菱形
</div>

<div class="component-box box-in" markdown="1">
## 输入

**X**  
X 尺寸

**Y**  
Y 尺寸

**Insets**  
从边缘向内缩进的量

**Angles**  
边的角度
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curves**  
单独的曲线

**Joined**  
连接的曲线

**Points**  
角点
</div>

</div>
