# Polygon

<img width="200" height="200" alt="Polygon component icon" src="/DesignTools/images/Polygon_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
基于半径和边数创建正多边形，可从菜单中选择可自定义的角类型。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**C2 Corners**  
每个角的平滑混合半径

**Arc Corners**  
每个角的简单圆弧半径

**Chamfered Corners**  
平边代替圆弧

**C2 Arc Corners**  
在每个角产生模仿圆弧的 C2 平滑半径
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Radius**  
构造圆的半径

**Sides**  
边数

**Fillet**  
角圆角

**Blend**  
角圆角混合

**Match**  
匹配构造圆的尺寸
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curves**  
单独的曲线

**Joined**  
连接的曲线

**Circle**  
构造圆

**Centres**  
圆角中心

**Points**  
说明
</div>

</div>
