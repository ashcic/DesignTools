# Asterisk

<img width="200" height="200" alt="Asterisk component icon" src="../../images/Asterisk_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
创建具有多个辐射臂的星号状形状。您可以自定义总体半径、每个臂的宽度和臂的总数。该组件提供三种不同风格：**Flat**、**Arched** 和 **Round**。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**Flat**  
每个臂创建平端

**Arched**  
每个臂创建与构造圆半径匹配的拱形端

**Round**  
每个臂创建完全圆角端
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Radius**  
包含圆的半径

**Width**  
每个臂的宽度

**Number of Arms**  
臂的数量
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curves**  
作为单独曲线的星号

**Joined**  
作为连接曲线的星号

**Points**  
控制点
</div>

</div>
