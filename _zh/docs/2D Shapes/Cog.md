# Cog

<img width="200" height="200" alt="Cog component icon" src="../../images/Cog_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
基于圆形基础生成齿轮状几何体（齿轮）。适用于管侧面的线性纹理，如相机镜头调节部件。

使用 *Radius* 和 *Segments* 确定基本形状，然后调整 *Angle*、*Offset* 和 *Extend* 以微调齿厚和间距。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**Angled**  
具有可调倾斜侧面的齿

**Square sides**  
垂直于圆的矩形齿

**Flat sides**  
简单平边齿

**Right-Angled**  
特定直角朝向的齿
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Radius**  
包含圆的半径

**Segments**  
齿数

**Angle**  
齿侧面角度

**Offset**  
控制齿大小

**Extend**  
改变齿宽度
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curves**  
单独的曲线

**Joined**  
连接的曲线

**Points**  
控制点

**Notes**  
如何使用此工具的说明
</div>

</div>
