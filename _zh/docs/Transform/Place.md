# Place

<img width="200" height="200" alt="Place component icon" src="/DesignTools/images/Place_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
将几何体从 XY 平面上的当前位置移动到目标平面，匹配目标平面的方向和位置。
</div>
<div class="component-box box-desc" markdown="1">
</div>
<div class="component-box box-in" markdown="1">
## 输入

**Geometry**  
说明

**Plane**  
要移动几何体到的平面
</div>

<div class="component-box box-out" markdown="1">
## 输出

**geometry**  
移动后的几何体

**Transform**  
向量移动 - 可输入到 Move 组件中以移动其他几何体
</div>
