# Revolve

<img width="200" height="200" alt="Revolve component icon" src="../../../images/Revolve_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
将轮廓曲线绕线旋转以创建 3D 曲面或实体。旋转角度可调，并可选择封盖结果几何体的起点和终点。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**start_cap**  
在曲线起点端添加封盖

**end_cap**  
在曲线终点端添加封盖
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Curves**  
轮廓曲线

**Angle**  
轮廓旋转的角度

**Axis**  
曲面绕其旋转的直线
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Revolve**  
最终曲面

**joined**  
连接的最终曲面
</div>

</div>
