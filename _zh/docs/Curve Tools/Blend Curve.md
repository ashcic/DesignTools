# Blend_Curve

<img width="200" height="200" alt="Blend Curve component icon" src="../../../images/Blend_Curve_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
在两点之间创建平滑曲线，可精确控制每个端点的方向。

**Orientation：** 为曲线的起点和终点定义特定方向（向量）和旋转（角度）。
**Optional Curves：** 混合到现有曲线以匹配特定半径或形状。
**Blend Influence：** 控制起点和终点设置对结果整体曲率的影响程度。
</div>
<div class="component-box box-desc" markdown="1">
</div>
<div class="component-box box-in" markdown="1">
## 输入

**Start Point**  
起点

**End Point**  
终点

**Start Vector**  
起点向量

**End Vector**  
终点向量

**Angle at Start**  
起点角度

**Angle at End**  
终点角度

**Start Curve**  
用于混合的可选曲线

**End Curve**  
用于混合的可选曲线

**Start Blend**  
起点向量的影响

**End Blend**  
终点向量的影响
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curve**  
混合曲线
</div>

</div>
