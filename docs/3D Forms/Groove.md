# Groove

#### Create a groove or nipple shape

Creates a 3D groove or nipple shape. You can adjust the dimensions of the profile using Length, Radius, and Depth, while Fillet and Blend parameters allow for smoothed edges.

___

### Inputs
**Length**
Length of the groove

**Radius**
Width of the groove

**Depth**
Depth of the groove

**Fillet**
Edge fillet

**Blend**
Blend of the edge fillet

___

### Outputs
**Brep**
The final brep

**Curves**
Edge curves

**Profile**
Profile cuves
