# Revolve

Created revolved surfaces from profile curves

___

### Inputs
**Curves**
Profile curves

**Angle**
The angle through which the profile is revolved

**Axis**
Line around which the surfaces are revolved

___

### Outputs
**Revolve**
Final surfaces

**joined**
Joined final surfaces
