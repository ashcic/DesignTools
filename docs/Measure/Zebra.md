# Zebra

#### Check surface curvature

Visualizes zebra stripes over a surface to help analyze curvature and smoothness. The size and rotation of the stripes can be adjusted using the scale and rotate inputs.

___

### Inputs
**Brep**
Input Brep

**scale**
Size of the stripes

**Rotate**
Rotate the stripes
