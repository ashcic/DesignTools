# Dimensions

<img width="200" height="200" alt="Dimensions component icon" src="../../images/Dimensions_400.png" />


Get the Bounding Box Dimensions of geometry as X, Y and Z values.

___

## Inputs

**Geometry**
Input geometry to measure

___

## Outputs

**X**
The X Dimension

**Y**
The Y Dimension

**Z**
The Z Dimension
