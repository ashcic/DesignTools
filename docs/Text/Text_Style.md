# Text Style

Set the style for the Text node

___

### Inputs
**Font**
Font

**Text Case**
Normal, Upper case, Lower case, Title case

**Height**
Text height

**Horizontal Align**
0,1,2 for left, centre, right aligned

**Vertical Align**
0,1,2 for above, centred, below the line

___

### Outputs
**style**
Style for the Text node
