# Text

<img width="200" height="200" alt="Text component icon" src="../../images/Text_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
Converts a text into geometry based on a specified style.
</div>
<div class="component-box box-desc" markdown="1">
</div>
<div class="component-box box-in" markdown="1">
## Inputs

**Text**  
This is the text to convert

**Plane**  
Set a plane to position the text

**Style**  
Use a 'Text Style' node to generate the string to control the text style
</div>

<div class="component-box box-out" markdown="1">
## Outputs

**Curves**  
Curves from the text

**Hatches**  
Hatches can be exported to Adobe Illustrator as solid shape

**Bounds**  
A rectangle around the text which is useful for laying out the text
</div>

</div>
