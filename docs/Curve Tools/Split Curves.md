# Split Curves

<img width="200" height="200" alt="Split Curves component icon" src="../../images/Split_Curves_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
Splits a set of curves using one or more cutting planes. The resulting segments are organized into branches based on the sections created by those planes.
</div>

<div class="component-box box-menu" markdown="1">
## Menu Options

**Close**  
Add a line to close the curves into a loop
</div>

<div class="component-box box-in" markdown="1">
## Inputs

**Curves**  
Curves to be split

**Planes**  
Cutting planes

**Fillet**  
Fillet at split

**Blend**  
Blend of fillets

**Gap**  
Add a gap size between splits
</div>

<div class="component-box box-out" markdown="1">
## Outputs

**Curves**  
Split curves by branch

**Joined**  
Joined curve results

**Points**  
Intersection points
</div>

</div>
