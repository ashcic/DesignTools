# Fillet Curves

Applies blend fillets to a set of curves

___

### Inputs
**Curves**
The main curves

**Radii**
List of the radius values for each fillet

**Blends**
List of the blend values for each fillet

___

### Outputs
**Curves**
Individual curves

**Joined**
Joined curves
