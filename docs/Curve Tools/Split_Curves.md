# Split Curves

Split cuves with one or more planes

___

### Inputs
**Curves**
Curves to be split

**Planes**
Cutting planes

**Fillet**
Fillet at split

**Blend**
Blend of fillets

**Gap**
Add a gap size between splits

___

### Outputs
**Curves**
Split curves by branch

**Joined**
Joined curve results

**Points**
Intersection points

___
