# Blend_Curve

Create and control a Blend Curve between points

___

### Inputs
**Start Point**
Point at the start

**End Point**
Point at the end

**Start Vector**
Vector at the start

**End Vector**
Vector at the end

**Angle at Start**
Angle at the start

**Angle at End**
Angle at the end

**Start Curve**
Optional Curve to blend with

**End Curve**
Optional Curve to blend with

**Start Blend**
Influence of the start vector

**End Blend**
Influence of the end vector

___

### Outputs
**Curve**
Blend Curve
