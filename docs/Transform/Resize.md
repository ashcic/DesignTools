# Resize

Change the size of the geometry

___

### Inputs
**Geometry**
Geometry to resize

**refs**
The geometry to use as a reference

**x**
X target sizing

**y**
Y target sizing

**z**
Z target sizing

___

### Outputs
**Geometry**
Resized geometry

**Planes**
Transformation base planes

**Scales**
Applied scale factors

**Moves**
The translation vectors used to move the geometry

**Notes**
A description of how to use this tool
