# Place

<img width="200" height="200" alt="Place component icon" src="../../images/Place_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
Moves geometry from its current location on the XY plane onto a target plane, matching the orientation and position of the target plane.
</div>

<div class="component-box box-in" markdown="1">
## Inputs

**Geometry**  
Description

**Plane**  
Plane to move geometry to
</div>

<div class="component-box box-out" markdown="1">
## Outputs

**geometry**  
Moved Geometry

**Transform**  
Vector Move - useful for inpuptting into a Move component and moving other geometry
