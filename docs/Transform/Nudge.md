# Nudge

<img width="200" height="200" alt="Nudge component icon" src="../../images/Nudge_400.png" />


Nudge geometry in small amounts in X, Y and Z.  This is useful for making precise manual adjustments to the position of objects.

___

## Inputs

**Geometry**
Geometry to nudge

___

## Outputs

**Geometry**
Nudged geometry

___

## Menu Options

**X Axis**
X Direction

**Y Axis**
Y Direction

**Z Axis**
Z Direction
