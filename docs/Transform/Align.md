# Align

<img width="200" height="200" alt="Align component icon" src="../../images/Align_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
Align the geometry in the X, Y or Z dimensions.  Arranges multiple geometries into a target area defined by reference geometry, a center point, or their own combined boundaries. You can choose to align the objects as a single group or individually while adjusting the layout size with insets and specific X, Y, Z positions.
</div>

<div class="component-box box-menu" markdown="1">
## Menu Options

**Union**  
Treat the geometry as a single group of objects

**Centre**  
Align around the Origin
</div>

<div class="component-box box-in" markdown="1">
## Inputs

**Geometry**  
Main Geometry

**Reference**  
The geometry to use as a reference

**X**  
0, 1 or 2 sets the position to left, centre or right

**Y**  
0, 1 or 2 sets the position to front, centre or back

**Z**  
0, 1 or 2 sets the position to bottom, centre or top

**Inset**  
Inset the geometry from the reference point
</div>

<div class="component-box box-out" markdown="1">
## Outputs

**Geometry**  
The updated geometry

**Moves**  
The translation vectors used to move the geometry

**Notes**  
A description of how to use this tool
</div>

</div>
