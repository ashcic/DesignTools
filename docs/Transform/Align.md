# Align

Align the geometry in the X, Y or Z dimensions

___

### Inputs
**Geometry**
Main Geometry

**Reference**
The geometry to use as a reference

**X**
0, 1 or 2 sets the position to left, centre or right

**Y**
0, 1 or 2 sets the position to front, centre or back

**Z**
0, 1 or 2 sets the position to bottom, centre or top

**Inset**
Description

___

### Outputs
**Geometry**
The updated geometry

**Moves**
The translation vectors used to move the geometry

**Notes**
A description of how to use this tool
