# Centre

Finds the centre of geometry

___

### Inputs
**Geometry**
The main geometry

**Location**
0-26 which gets other key points on the boundingbox of the geometry

___

### Outputs
**Geometry**
The updated geometry

**Moves**
The translation vectors used to move the geometry
