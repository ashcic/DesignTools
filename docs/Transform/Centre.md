# Centre

<img width="200" height="200" alt="Centre component icon" src="../../images/Centre_400.png" />


### Finds the centre of geometry

Finds the centre of geometry based on bounding boxes.<br><br>• **Union:** Get the centre of all items together as one group.<br>• **Individual:** Get the centres of each item independently.

___

## Inputs

**Geometry**
The main geometry

**Location**
0-26 which gets other key points on the boundingbox of the geometry

___

## Outputs

**Geometry**
The updated geometry

**Moves**
The translation vectors used to move the geometry

___

## Menu Options

**Union**
Treat the geometry as a single group of objects

**Individual**
Treat as individual objects
