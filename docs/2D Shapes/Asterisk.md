# Asterisk

Creates an Astrix shape

___

### Inputs
**Radius**
Radius of the containing circle

**Width**
Width of each arm

**Number of Arms**
Number of arms

___

### Outputs
**curves**
Individual curves

**joined**
Joined curves

**Points**
Control Points
