# Triangle

Creates a triangle that can be oriented in different directions.  Use an integer number slider in the 'orient' input to change the direction it is pointing

___

## Inputs

**Radius**
The radius of the construction circle

**Orient**
Direction that the triangle points

___

## Outputs

**curves**
Individual curves

**joined**
Joined curves

**points**
Control Points
