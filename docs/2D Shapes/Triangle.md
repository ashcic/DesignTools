# Triangle

<img width="200" height="200" alt="Triangle component icon" src="../../images/Triangle_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
Creates a triangle that can be oriented in different directions.  Use an integer number slider in the 'orient' input to change the direction it is pointing
</div>

<div class="component-box box-in" markdown="1">
## Inputs

**Radius**  
The radius of the construction circle

**Orient**  
Direction that the triangle points
</div>

<div class="component-box box-out" markdown="1">
## Outputs

**curves**  
Individual curves

**joined**  
Joined curves

**points**  
Control Points
</div>

</div>
