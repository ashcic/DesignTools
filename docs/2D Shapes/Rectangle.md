# Rectangle

Description

___

### Inputs
**X**
The X Dimension

**Y**
The Y Dimension

**Radii**
You can add multiple dimensions for multiple radii

**Blends**
Control how much the corners blend into the sides

___

### Outputs
**Curves**
The rectangle as separate curves

**Joined**
The rectangle as joined curves

**Points**
The corner points

**Notes**
A description of how to use this tool
