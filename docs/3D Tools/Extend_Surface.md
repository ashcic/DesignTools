# Extend Surface

#### Extend a surface from an edge

Extends an existing surface outward from one or more of its edges by a specified distance. You can choose to extend specific edges or all sides simultaneously.

___

### Inputs
**Surface**
The main surface

**Length**
Length of the extension to the surface

**Edge**
Choose which edge to extend

___

### Outputs
**Surface**
The extended surface

**Notes**
A description of how to use this tool
