# Split Brep

Split a brep with a plane

___

### Inputs
**Brep**
Geometry to be split

**Plane**
Orientation of the split

**Fillet**
Radius of transition fillet

**Blend**
Blend factor (0-1)

**Flip**
Flip final geometry orientation

___

### Outputs
**Breps**
Resulting joined Breps

**Curves**
Intersection/split curves
