# Text Style

<img width="200" height="200" alt="Text Style component icon" src="../../images/Text_Style_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
为 Text 节点定义样式属性。使用此组件设置：

**Font** 字体系列
**Text Case** (Normal, Upper, Lower, 或 Title case) 大小写
**Height** 文本高度
**Alignment** (水平和垂直位置) 对齐

输出提供格式化的样式字符串，供对应的 Text 组件使用。
</div>
<div class="component-box box-desc" markdown="1">
</div>
<div class="component-box box-in" markdown="1">
## 输入

**Font**  
字体

**Text Case**  
正常、大写、小写、首字母大写

**Height**  
文本高度

**Horizontal Align**  
0,1,2 分别为左对齐、居中、右对齐

**Vertical Align**  
0,1,2 分别为上方、居中、基线下方
</div>

<div class="component-box box-out" markdown="1">
## 输出

**style**  
Text 节点的样式
</div>

</div>
