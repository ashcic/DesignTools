# Re-Rectangle

<img width="200" height="200" alt="Re-Rectangle component icon" src="../../images/Re_Rectangle_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
基于参考几何体构建矩形、胶囊形状或圆角盒。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**C2 Corners**  
每个角的平滑混合半径

**Arc Corners**  
每个角的简单圆弧半径

**Chamfered Corners**  
平边代替圆弧

**C2 Arc Corners**  
在每个角产生模仿圆弧的 C2 平滑半径
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Geometry**  
主要几何体

**Offset**  
说明

**Radii**  
可为多个半径添加多个尺寸

**Blends**  
控制角与边的混合程度

**Keeps**  
哪些角应保持原样

**Match**  
说明
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Curves**  
初始矩形作为单独曲线

**Offsets**  
偏移矩形作为单独曲线

**Joined**  
作为连接曲线的矩形

**Points**  
角点

**Values**  
初始几何体的各种尺寸

**Notes**  
如何使用此工具的说明
</div>

</div>
