# Align

<img width="200" height="200" alt="Align component icon" src="../../images/Align_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
在 X、Y 或 Z 维度上对齐几何体。将多个几何体排列到由参考几何体、中心点或其自身组合边界定义的目标区域中。您可以选择将对象作为单个组对齐或单独对齐，同时使用内缩和特定的 X、Y、Z 位置调整布局大小。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**Union**  
将几何体视为单个对象组

**Centre**  
围绕原点对齐
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Geometry**  
主几何体

**Reference**  
用作参考的几何体

**X**  
0、1 或 2 将位置设置为左、中或右

**Y**  
0、1 或 2 将位置设置为前、中或后

**Z**  
0、1 或 2 将位置设置为底、中或顶

**Inset**  
从参考点向内缩进几何体
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Geometry**  
更新后的几何体

**Moves**  
用于移动几何体的平移向量

**Notes**  
如何使用此工具的说明
</div>

</div>
