# Extend Surface

<img width="200" height="200" alt="Extend Surface component icon" src="../../images/Extend_Surface_400.png" />

<div class="component-grid" markdown="1">

<div class="component-box box-desc" markdown="1">
按指定距离或比例从一条或多条边向外延伸现有曲面。您可以选择延伸特定边或同时延伸所有边。
</div>

<div class="component-box box-menu" markdown="1">
## 菜单选项

**Use Simple Domain Extension**  
按比例而非长度延伸曲面
</div>

<div class="component-box box-in" markdown="1">
## 输入

**Surface**  
主要曲面

**Length**  
曲面延伸长度

**Edge**  
选择要延伸的边
</div>

<div class="component-box box-out" markdown="1">
## 输出

**Surface**  
延伸后的曲面

**Notes**  
如何使用此工具的说明
</div>

</div>
